# 🎧 ЗвукЛист - Полный функционал Spotify

Музыкальная платформа с полным функционалом Spotify на 100% русском языке.

## 🎯 Новые функции (Spotify функционал)

### 📊 Боковое меню навигации
- 🏠 **Главная** - главный экран с плеером и рекомендациями
- 🔍 **Поиск** - поиск по названию и исполнителю
- ⭐ **Рекомендации** - персональные рекомендации на основе истории
- 🎵 **Категории** - музыка по жанрам
- 📋 **Плейлисты** - создание и управление плейлистами
- ⏱️ **История** - все прослушанные треки
- ❤️ **Избранное** - сохраненные любимые треки
- 🌙 **Тема** - переключение светлой/тёмной темы

### 🎵 Музыкальный функционал

#### Плеер
- ▶️ Play/Pause
- ⏮️ Назад (или перемотка в начало если начато)
- ⏭️ Вперёд
- 🔁 Повтор (выкл → один трек → все треки)
- 🔀 Перемешивание (случайный порядок)
- 🔊 Регулировка громкости
- 📊 Прогресс-бар с визуализацией

#### Информация о треке
- 🎨 Живая обложка (вращается при проигрывании)
- 🎶 Эффект эквалайзера
- 📝 Название и исполнитель
- ⏱️ Текущее время и длительность

### 📋 Плейлисты
- ➕ Создание новых плейлистов
- 📝 Название и описание плейлиста
- 🎶 Добавление/удаление треков
- 💾 Автоматическое сохранение
- 📊 Количество треков в плейлисте

### ⭐ Рекомендации и похожие треки
- 🤖 Интеллектуальные рекомендации на основе:
  - Истории прослушивания
  - Жанра текущего трека
  - Избранных треков
- 🎶 Похожие треки по жанру
- 🔄 Обновление в реальном времени

### 🔍 Поиск
- 🎯 Быстрый поиск в заголовке
- 🏷️ Поиск по названию и исполнителю
- 📌 Выпадающие результаты
- ⌨️ Клавиатурный ввод

### 🎵 Категории и жанры

## 🎯 Как добавить большое количество треков

### Шаг 1: Подготовка JSON
```bash
# Используйте JSON генератор или напишите скрипт для создания:
[
  {
    "id": 1,
    "title": "Песня 1",
    "artist": "Артист 1",
    "src": "assets/music/track1.mp3",
    "cover": "assets/covers/cover1.jpg",
    "duration": 200,
    "genre": "Электроника",
    "year": 2024
  },
  // ... остальные треки
]
```

### Шаг 2: Добавить обложки
- Положите JPG/PNG в `assets/covers/`
- Рекомендуемый размер: 300x300px
- Назовите: cover1.jpg, cover2.jpg и т.д.

### Шаг 3: Добавить музыку
- Положите MP3 в `assets/music/`
- Назовите: track1.mp3, track2.mp3 и т.д.

### Шаг 4: Обновить music.json
- Скопируйте структуру из примера
- Убедитесь, что пути совпадают
- Проверьте синтаксис JSON

## 🐛 Решение проблем

### Треки не загружаются
- Проверьте консоль браузера (F12)
- Убедитесь, что music.json правильный синтаксис
- Проверьте пути к файлам

### Рекомендации не показываются
- Слушайте треки (добавятся в историю)
- Прежде слушайте из разных жанров
- Рекомендации исходят из истории

### Плейлисты не сохраняются
- Проверьте, что localStorage включен в браузере
- Очищайте кэш и перезагружайте
- Попробуйте приватный/инкогнито режим

### Поиск не работает
- Убедитесь, что введены русские или английские буквы
- Поиск чувствителен к регистру (А и а - одно)
- Просто введите часть названия

## 📚 Примеры использования

### Программный выбор трека
```javascript
// В консоли браузера
player.selectTrack(1);      // Выбрать трек с ID 1
player.play();              // Начать проигрывание
```

### Добавление в избранное
```javascript
player.toggleFavorite();    // Переключить избранное
player.renderFavorites();   // Обновить список
```

### Создание плейлиста
```javascript
// Используйте кнопку в интерфейсе или:
const newPlaylist = {
    id: 99,
    name: "Мой плейлист",
    description: "Описание",
    tracks: [1, 2, 3],
    cover: "assets/covers/cover1.jpg"
};
player.userPlaylists.push(newPlaylist);
```
