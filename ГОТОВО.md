🚀 КАК ИСПОЛЬЗОВАТЬ

ЗАПУСК:
1. Откройте VS Code
2. Откройте папку zvuklist
3. Нажмите правой кнопкой на index.html
4. "Open with Live Server"
5. ✅ Проект загрузится в браузере!

ИЛИ просто:
1. Откройте index.html двойным кликом
2. ✅ Откроется в браузере!

ДОБАВИТЬ МУЗЫКУ:
1. Положите MP3 в assets/music/
2. Положите JPG обложку в assets/covers/
3. Отредактируйте music.json
4. Обновите браузер (F5)
5. ✅ Музыка появится!

ИЗМЕНИТЬ ЦВЕТА:
1. Откройте style.css
2. Найдите :root { ... }
3. Измените --accent-color: #ff4d6d;
4. Сохраните (Ctrl+S)
5. Обновите браузер (F5)
6. ✅ Цвета изменились!
