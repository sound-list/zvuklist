# ⚡ Краткая шпаргалка ЗвукЛист

## 🎯 Что это?
**ЗвукЛист** - музыкальная платформа на 100% русском языке. Никакой БД, всё работает в браузере.

## 📁 Структура
```
zvuklist/
├── index.html          ← Откройте это (HTML)
├── style.css           ← Стили (красиво)
├── script.js           ← Функциональность (JS)
├── music.json          ← Список воспроизведения
├── assets/
│   ├── music/          ← Положите MP3 сюда
│   └── covers/         ← Положите JPG сюда
└── README.md           ← Полная документация
```

## 🚀 Запуск за 30 секунд

### Windows (Live Server в VS Code)
1. **Правый клик** на `index.html`
2. **"Open with Live Server"**
3. ✅ Готово!

### Или просто откройте в браузере
1. **Правый клик** на `index.html`
2. **"Open with"** → браузер
3. ✅ Готово!

## 🎵 Добавить музыку за 1 минуту

### 1. Положите файлы
- MP3 → `assets/music/песня.mp3`
- Обложка JPG → `assets/covers/обложка.jpg`

### 2. Отредактируйте `music.json`
```json
[
  {
    "title": "Моя песня",
    "artist": "Мой артист",
    "src": "assets/music/песня.mp3",
    "cover": "assets/covers/обложка.jpg",
    "duration": 240
  }
]
```

### 3. Обновите браузер
Готово! Песня появится в списке.

## 🎨 Изменить цвета за 2 минуты

Откройте `style.css` и найдите начало файла:

```css
:root {
    --primary-color: #1a1a40;     ← Основной фон
    --secondary-color: #3a0ca3;   ← Градиент
    --accent-color: #ff4d6d;      ← Кнопки
}
```

**Примеры цветов:**
- Зелёный: `#00ff7f`
- Синий: `#00d4ff`
- Оранжевый: `#ff7a3d`
- Золотой: `#ffb700`

## 🐛 Проблемы

### Треки не видны
❌ Проверьте `music.json` на ошибки синтаксиса
✅ Используйте [jsonlint.com](https://jsonlint.com/)

### Нет звука
❌ Проверьте путь в `src`: `assets/music/file.mp3`
✅ Убедитесь, что файл действительно там

### Обложка не показывается
❌ Убедитесь, что файл JPG/PNG существует
✅ Проверьте путь в `cover`

## 💡 Быстрые трюки

### Очистить все сохранённые данные
Откройте консоль браузера (F12) и введите:
```javascript
localStorage.clear();
```

### Добавить свой шрифт
В `style.css` измените:
```css
font-family: 'Arial', sans-serif;  /* Измените 'Arial' */
```

### Сделать кнопки больше
В `style.css` найдите `.control-btn` и измените:
```css
padding: 12px 16px;  ← Увеличьте цифры
font-size: 14px;     ← Увеличьте шрифт
```

## 🔐 Структура данных

**music.json** - каждый трек имеет:
- `title` (строка) - название на русском
- `artist` (строка) - исполнитель на русском
- `src` (строка) - путь к MP3
- `cover` (строка) - путь к обложке JPG/PNG
- `duration` (число) - длительность в секундах (можно узнать в плеере)

## 🎓 Для разработчиков

Главный класс в `script.js`:
```javascript
class MusicPlayer {
    play()           // Воспроизведение
    pause()          // Пауза
    nextTrack()      // Следующий
    prevTrack()      // Предыдущий
    toggleRepeat()   // Повтор
    toggleShuffle()  // Перемешивание
    toggleFavorite() // Избранное
}
```

Доступ из консоли:
```javascript
musicPlayer.play()           // Включить
musicPlayer.pause()          // Выключить
musicPlayer.nextTrack()      // Следующий
musicPlayer.currentTrackIndex = 0; // Переключиться на трек 0
```
