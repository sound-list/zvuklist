## 🚀 Добавление своей музыки

### Шаг 1: Подготовка файлов

**MP3 файлы:**
```
zvuklist/assets/music/
├── track1.mp3
├── track2.mp3
└── ... (столько сколько хотите)
```

**Обложки (JPG/PNG):**
```
zvuklist/assets/covers/
├── cover1.jpg
├── cover2.jpg
└── ... (для каждого трека)
```

### Шаг 2: Обновите music.json

Откройте `music.json` и добавьте треки:

```json
{
  "tracks": [
    {
      "id": 1,
      "title": "Название песни",
      "artist": "Имя артиста",
      "src": "assets/music/track1.mp3",
      "cover": "assets/covers/cover1.jpg",
      "duration": 245,
      "genre": "Электроника",
      "year": 2024
    }
  ],
  "genres": [
    {
      "name": "Электроника",
      "icon": "⚡",
      "color": "#00d4ff"
    }
  ],
  "playlists": []
}
```

### Шаг 3: Обновите браузер

Нажмите **F5** в браузере - ваши треки появятся!

## 🎨 Быстрая кастомизация

### Изменить основной цвет (розовый на синий)

Откройте `style_spotify.css` и найдите:

```css
:root {
  --accent-color: #ff4d6d;  /* Розовый */
}
```

Замените на:

```css
:root {
  --accent-color: #0066ff;  /* Синий */
}
```

Больше опций смотрите в [CUSTOMIZATION.md](CUSTOMIZATION.md)

## 💡 Примеры управления из консоли

Откройте консоль браузера **F12** и попробуйте:

```javascript
// Информация о текущем треке
console.log(player.currentTrackIndex);

// Все треки
console.log(player.allTracks);

// Запустить рекомендации
player.renderRecommendations();

// Переключить тему
player.toggleTheme();

// Установить громкость (от 0 до 1)
player.audioPlayer.volume = 0.5;

// Очистить все данные
localStorage.clear();
```
