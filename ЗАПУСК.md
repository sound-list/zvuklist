# 📖 Инструкция по запуску ЗвукЛист

## ✅ Что уже создано

Полная музыкальная платформа на **100% русском языке** со следующей структурой:

```
zvuklist/
├── index.html              ✅ HTML структура (весь текст на русском)
├── style.css               ✅ Стили и дизайн (glassmorphism)
├── script.js               ✅ Функциональность (весь код на русском)
├── music.json              ✅ Список треков (примеры на русском)
├── README.md               ✅ Документация (полностью на русском)
├── assets/
│   ├── music/              📁 Папка для MP3 файлов
│   └── covers/             📁 Папка для обложек (JPG/PNG)
└── ЗАПУСК.md               📄 Этот файл
```

## 🚀 Быстрый старт

### Для Windows 10/11

#### Способ 1: Live Server в VS Code (Рекомендуется)

1. **Откройте VS Code**
   - Нажмите `Win + R` и введите `code`
   - Или откройте VS Code из меню Пуск

2. **Откройте папку проекта**
   - `Ctrl + K` → `Ctrl + O`
   - Выберите папку `zvuklist`
   - Нажмите "Select Folder"

3. **Установите Live Server** (если нет)
   - Нажмите `Ctrl + Shift + X` (Extensions)
   - Напишите "Live Server"
   - Установите расширение от Ritwick Dey

4. **Запустите проект**
   - Откройте файл `index.html`
   - Нажмите правой кнопкой → "Open with Live Server"
   - Автоматически откроется браузер на `http://localhost:5500`

#### Способ 2: Прямое открытие в браузере

1. Откройте проводник (Windows + E)
2. Перейдите в папку `zvuklist`
3. Дважды кликните на `index.html`
4. Платформа откроется в вашем браузере по умолчанию

#### Способ 3: Из командной строки

```powershell
# Откройте PowerShell в папке проекта (Shift + Right Click → Open PowerShell)
# Или навигируйте: cd D:\Рабочий стол\zvuklist

# Запустите локальный сервер Python (если установлен):
python -m http.server 8000

# Или используйте node.js (если установлен):
npx http-server

# Потом откройте браузер на указанном адресе (обычно http://localhost:8000)
```

## 🎵 Добавление своей музыки

### Шаг 1: Подготовка файлов

1. **Подготовьте MP3 файлы**
   - Положите MP3 файлы в папку `zvuklist/assets/music/`
   - Убедитесь, что файлы в формате `.mp3`

2. **Подготовьте обложки**
   - Положите изображения обложек в папку `zvuklist/assets/covers/`
   - Формат: JPG или PNG
   - Рекомендуемый размер: 300×300px

### Шаг 2: Обновление music.json

Откройте файл `music.json` в VS Code и отредактируйте список:

```json
[
  {
    "title": "Русское название трека",
    "artist": "Русское имя исполнителя",
    "src": "assets/music/файл.mp3",
    "cover": "assets/covers/обложка.jpg",
    "duration": 240
  }
]
```

**Обязательные поля:**
- `title` - название песни (русский текст)
- `artist` - имя певца/группы (русский текст)
- `src` - путь к MP3 файлу
- `cover` - путь к файлу обложки
- `duration` - длительность в секундах

### Пример с несколькими треками:

```json
[
  {
    "title": "Волшебная ночь",
```

## 🎨 Кастомизация

### Изменение цветов

Откройте `style.css` и найдите раздел `:root`:

```css
:root {
    --primary-color: #1a1a40;        /* Тёмно-синий фон */
    --secondary-color: #3a0ca3;      /* Синий градиент */
    --accent-color: #ff4d6d;         /* Розовый (кнопки) */
    --accent-hover: #ff758f;         /* Светлый розовый */
    --text-color: #ffffff;           /* Белый текст */
    --text-secondary: rgba(255, 255, 255, 0.7);  /* Полупрозрачный текст */
}
```

**Примеры красивых цветовых схем:**

**Зелёный лес:**
```css
--primary-color: #0d3a0d;
--secondary-color: #2d6b2d;
--accent-color: #00ff88;
--accent-hover: #00ffaa;
```

**Оранжевый закат:**
```css
--primary-color: #3d2a1f;
--secondary-color: #6d4a2f;
--accent-color: #ff7a3d;
--accent-hover: #ff9a5d;
```

**Фиолетовый свет:**
```css
--primary-color: #2d0d3d;
--secondary-color: #5d0d8d;
--accent-color: #d46bff;
--accent-hover: #e889ff;
```

### Изменение шрифта

Найдите строку `font-family` в `style.css`:

```css
font-family: 'Segoe UI', 'Roboto', sans-serif;
```

**Популярные шрифты:**
- `'Arial', sans-serif` - классический
- `'Georgia', serif` - элегантный
- `'Courier New', monospace` - программистский
- `'Comic Sans MS', cursive` - весёлый

Все данные сохраняются в браузер автоматически:
- ✅ Избранные треки
- ✅ Текущий трек
- ✅ Громкость
- ✅ Тема оформления
- ✅ Режим повтора

Откройте консоль браузера (F12) и введите:
```javascript
localStorage.clear();
```

## 🐛 Решение проблем

### Треки не видны в списке
- ❌ Проверьте синтаксис JSON (нет лишних запятых)
- ✅ Используйте [JSON валидатор](https://jsonlint.com/)

### Обложки не загружаются
- ❌ Проверьте путь до файла
- ✅ Убедитесь, что файл реально существует в папке
- ✅ Используйте только JPG или PNG

### Звука нет
- ❌ Проверьте громкость (🔊)
- ✅ Убедитесь, что MP3 файл не повреждён
- ✅ Попробуйте открыть MP3 в браузере отдельно

### "Не удалось загрузить список треков"
- ❌ `music.json` содержит ошибки синтаксиса
- ✅ Откройте консоль браузера (F12) и посмотрите ошибку
- ✅ Исправьте JSON формат
