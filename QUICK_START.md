# ⚡ Быстрый старт - Quick Start Guide

Начните использовать ЗвукЛист за 5 минут!

## 🚀 Способ 1: Самый быстрый (3 минуты)

### 1️⃣ Откройте файл

Right-click на `index.html` → **Открыть с помощью → VS Code**

Или если у вас VS Code уже открыт:
- **File → Open Folder → выберите `zvuklist`**

### 2️⃣ Запустите Live Server

В VS Code:
- Right-click на `index.html` → **Open with Live Server**

Или нажмите кнопку "Go Live" внизу справа

### 3️⃣ Готово! 🎉

Музыкальная платформа откроется в браузере.

## 🎵 Добавление своей музыки

### Шаг 1: Подготовка файлов

**Музыка (MP3):**
- Положите MP3 файлы в папку: `assets/music/`
- Переименуйте: `track1.mp3`, `track2.mp3` и т.д.

**Обложки (JPG/PNG):**
- Положите изображения в папку: `assets/covers/`
- Переименуйте: `cover1.jpg`, `cover2.jpg` и т.д.
- Размер: 300x300px (оптимально)

### Шаг 2: Обновэтель music.json

Откройте `music.json` и отредактируйте массив `"tracks"`:

```json
{
  "tracks": [
    {
      "id": 1,
      "title": "Название песни",
      "artist": "Имя артиста",
      "src": "assets/music/track1.mp3",
      "cover": "assets/covers/cover1.jpg",
      "duration": 260,
      "genre": "Электроника",
      "year": 2024
    }
  ],
  "genres": [],
  "playlists": []
}
```

### Шаг 3: Обновите браузер

```
F5 или Ctrl + R
```

**Готово!** Ваши треки будут видны в приложении.

## 🎨 Быстрое изменение внешнего вида

### Изменить цвет (розовый на синий)

Откройте `style_spotify.css` и найдите:

```css
:root {
  --accent-color: #ff4d6d;  /* Розовый */
}
```

Замените на:

```css
:root {
  --accent-color: #0066ff;  /* Синий */
}
```

Обновите браузер (F5) - цвет изменится!

### Выбор других цветов

```css
/* Красный */
--accent-color: #ff0000;

/* Зеленый */
--accent-color: #00ff00;

/* Фиолетовый */
--accent-color: #aa00ff;

/* Оранжевый */
--accent-color: #ff9900;

/* Кирпичный */
--accent-color: #e63946;
```

### Светлая тема по умолчанию

В `script_spotify.js` найдите в функции `constructor()`:

```javascript
// Измените эту строку:
document.body.className = 'dark-theme';

// На эту:
document.body.className = 'light-theme';
```

## 🎓 Частые действия

### Как создать плейлист?

1. Нажмите **📋 Плейлисты** в меню
2. Нажмите **➕ Создать плейлист**
3. Введите название и описание
4. Нажмите **Создать**

### Как добавить трек в плейлист?

1. Проигрывайте трек или выберите в списке
2. Нажмите **➕ В плейлист** в плеере
3. Выберите мой плейлист
4. Трек добавлен!

### Как найти трек?

1. Нажмите **🔍 Поиск** в меню
2. Введите название или артиста в строку поиска
3. Результаты появятся автоматически
4. Нажмите на результат для проигрывания

### Как просмотреть историю?

1. Нажмите **⏱️ История** в меню
2. Все прослушанные треки отсортированы по времени
3. Нажмите на трек для повторного проигрывания

### Как найти рекомендации?

1. Проиграйте несколько треков
2. Нажмите **⭐ Рекомендации** в меню
3. Система предложит похожие треки
4. Рекомендации обновляются при каждом проигрывании
