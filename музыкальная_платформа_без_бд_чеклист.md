# 🎧 Музыкальная платформа (без базы данных)

## 💡 Название платформы
**«ЗвукЛист»** — лёгкая веб-платформа для прослушивания музыки без использования базы данных.

---

## 🚀 Общая идея
Платформа работает на статических файлах (JSON / localStorage) без сервера базы данных.

---

## 📁 Структура проекта

```
project/
│── index.html
│── style.css
│── script.js
│── /assets
│    ├── /music
│    ├── /covers
│── music.json
```

---

## 🧩 Что нужно добавить

### 1. HTML (index.html)
- Аудиоплеер
- Список треков
- Кнопки управления (play/pause/next)
- Прогресс-бар
- Обложка трека

### 2. CSS (style.css)
- Градиентный фон
- Glassmorphism (размытие)
- Анимации кнопок
- Hover эффекты
- Адаптивность

### 3. JavaScript (script.js)
- Загрузка треков из `music.json`
- Управление аудио (play/pause)
- Переключение треков
- Прогресс трека
- localStorage для сохранения последнего трека

### 4. music.json
Пример структуры:
```json
[
  {
    "title": "Track 1",
    "artist": "Artist",
    "src": "assets/music/track1.mp3",
    "cover": "assets/covers/cover1.jpg"
  }
]
```

---

## 🎨 Дизайн (очень красивый)

### 🌈 Цвета
- Фон: градиент (фиолетовый → синий)
- Акцент: неоновый розовый
- Текст: белый / полупрозрачный

### ✨ Стиль
- Glass UI (blur + прозрачность)
- Скругления: 20px
- Тени: мягкие, не яркие

### 🧊 Пример CSS
```css
body {
  margin: 0;
  font-family: 'Segoe UI', sans-serif;
  background: linear-gradient(135deg, #1a1a40, #3a0ca3);
  color: white;
}

.player {
  backdrop-filter: blur(20px);
  background: rgba(255, 255, 255, 0.1);
  border-radius: 20px;
  padding: 20px;
  box-shadow: 0 8px 32px rgba(0,0,0,0.3);
}

button {
  background: #ff4d6d;
  border: none;
  padding: 10px 20px;
  border-radius: 10px;
  cursor: pointer;
  transition: 0.3s;
}

button:hover {
  transform: scale(1.1);
  background: #ff758f;
}
```

## 📦 Как запустить
1. Открыть проект в Visual Studio Code
2. Установить расширение Live Server
3. Запустить `index.html`
