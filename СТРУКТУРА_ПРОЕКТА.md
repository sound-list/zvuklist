## 📊 JavaScript Методы

### Управление воспроизведением
- `play()` - начать проигрывание
- `pause()` - остановить
- `togglePlay()` - переключить play/pause
- `nextTrack()` - следующий трек
- `prevTrack()` - предыдущий трек

### Управление состоянием
- `toggleRepeat()` - переключить режим повтора
- `toggleShuffle()` - переключить перемешивание
- `toggleFavorite()` - добавить/удалить из избранного
- `toggleTheme()` - переключить светлую/тёмную тему

### Загрузка и рендеринг
- `loadTracks()` - загрузить треки из music.json
- `loadCurrentTrack()` - загрузить текущий трек
- `renderTrackList()` - отрендерить список
- `renderFavorites()` - отрендерить избранное

### Утилиты
- `formatTime()` - преобразовать сек в MM:SS
- `setVolume()` - установить громкость
- `seek()` - перемотка трека
- `updateProgress()` - обновить прогресс-бар

### localStorage операции
- `saveFavorites()` - сохранить избранное
- `loadFavorites()` - загрузить избранное
- `saveCurrentTrack()` - сохранить текущий трек
- `loadTheme()` - загрузить сохранённую тему

## 🔄 Жизненный цикл приложения

```
1. HTML загружается
   ↓
2. CSS применяется (дизайн)
   ↓
3. JavaScript инициализируется
   ├─ Создаётся экземпляр MusicPlayer
   ├─ Загружаются треки из music.json
   ├─ Загружается избранное из localStorage
   ├─ Загружается тема из localStorage
   └─ Устанавливаются слушатели событий
   ↓
4. Плеер готов к использованию
   ├─ Пользователь может нажимать кнопки
   ├─ События срабатывают
   └─ DOM обновляется
   ↓
5. При закрытии браузера
   └─ localStorage сохраняет состояние
```

На мобильных устройствах:

- Уменьшается размер обложки
- ОптимизируетсяLayout

## 🎯 Последовательность запуска

1. **HTML парсится** → DOM готов
2. **CSS загружается** → стили применяются
3. **JS загружается** → код выполняется
4. **DOMContentLoaded** → создаётся MusicPlayer
5. **loadTracks()** → загружаются треки из JSON
6. **setupEventListeners()** → назначаются обработчики
7. **loadFavorites()** → восстанавливается избранное
8. **loadTheme()** → восстанавливается тема
9. **renderTrackList()** → отображается список
10. ✅ **Готово к использованию**
